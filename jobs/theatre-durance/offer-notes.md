# Théâtre Durance - Notes d'offre

## Statut

Base de travail pour une candidature liée au Théâtre Durance.

L'offre détaillée n'est pas encore documentée dans le dépôt. Les informations ci-dessous doivent être complétées à partir de l'annonce réelle avant rédaction finale.

## Informations à renseigner

- intitulé du poste ;
- type de contrat ;
- localisation ;
- missions ;
- compétences demandées ;
- publics concernés ;
- ton de l'offre ;
- éléments spécifiques au Théâtre Durance ;
- date limite ;
- pièces demandées.

## Points à vérifier

- niveau attendu en communication ;
- place du web ou des supports numériques ;
- lien avec les publics ;
- lien avec la programmation ou l'action culturelle ;
- contraintes administratives.
