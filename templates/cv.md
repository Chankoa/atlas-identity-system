# Template - CV

## En-tête

- Nom ;
- coordonnées ;
- lien portfolio ou site si disponible ;
- intitulé ciblé.

## Accroche

Utiliser une formulation issue de :

- `content/identity/positioning.md`
- `content/identity/bio-short.md`
- `profiles/[profil].md`

## Compétences

Sélectionner 4 à 6 compétences maximum selon le profil.

Chaque compétence doit être reliée à une expérience, un projet ou une formation documentée.

## Expériences

Pour chaque expérience :

- intitulé ;
- structure ou contexte ;
- période ;
- missions ;
- réalisations ;
- compétences mobilisées.

## Formation

Utiliser uniquement les informations confirmées dans `content/education/formations.md`.

## Règles

- Une page si possible.
- Hiérarchie claire.
- Aucun fait inventé.
- Adapter l'ordre des rubriques au profil ciblé.
