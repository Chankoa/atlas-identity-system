# Renderer - Adobe Express

## Rôle

Adobe Express sert à créer une version rapide, éditable et exportable de la candidature.

Il est adapté pour tester une mise en page de CV ou de lettre sans construire une maquette InDesign complète.

## Entrées Markdown

- `jobs/[candidature]/cv-[profil].md`
- `jobs/[candidature]/lettre-motivation.md`
- `jobs/[candidature]/email-candidature.md`
- `jobs/[candidature]/adobe-express-brief.md`
- `jobs/[candidature]/firefly-brief.md`

## Usage recommandé

1. Créer un document éditable dans Adobe Express.
2. Copier les textes depuis les fichiers Markdown.
3. Appliquer la palette et les principes du design system.
4. Garder les textes modifiables.
5. Exporter une version PDF uniquement après relecture.

## Sorties attendues

- Document Adobe Express éditable.
- PDF de relecture ou d'envoi, exporté manuellement.

## Limites

- Ne pas coller le CV comme image unique.
- Ne pas laisser Adobe Express réécrire le contenu.
- Ne pas remplacer les fichiers Markdown par la version graphique.
