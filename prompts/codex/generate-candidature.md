# Prompt - Générer une candidature

## Entrées nécessaires

- Offre ou notes dans `jobs/[candidature]/offer-notes.md`.
- Stratégie dans `jobs/[candidature]/strategy.md`.
- Profil cible dans `profiles/`.
- Template du livrable dans `templates/`.

## Instruction

À partir des sources disponibles dans le dépôt, produire une candidature adaptée au poste visé.

Respecter strictement les règles suivantes :

- ne pas inventer de fait personnel ;
- signaler les informations manquantes ;
- sélectionner uniquement les arguments utiles pour cette candidature ;
- garder un ton professionnel, sobre et éditorial ;
- relier communication, culture, web ou transmission seulement lorsque les sources le permettent.

## Sortie attendue

Produire :

- une synthèse de l'angle choisi ;
- une version du livrable demandé ;
- une liste des informations manquantes ;
- les fichiers sources utilisés.
