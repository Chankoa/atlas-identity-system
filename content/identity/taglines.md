# Taglines

- Communiquer, c'est créer les conditions d'une rencontre.
- Entre communication, création et transmission.
- Créer des liens entre les idées, les lieux et les personnes.

## Règles d'usage

- Utiliser une seule tagline par livrable.
- Privilégier la première pour une lettre ou un portfolio culturel.
- Privilégier la deuxième pour un profil généraliste.
- Privilégier la troisième pour une bio ou une introduction éditoriale.
