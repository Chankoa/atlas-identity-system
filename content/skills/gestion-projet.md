# Compétence - Gestion de projet

## Rôle dans le système

La gestion de projet permet de relier intention, organisation, livrables et contraintes.

## Usages possibles

- CV communication ;
- portfolio ;
- candidature à un poste polyvalent ;
- valorisation d'une activité freelance.

## Formulations mobilisables

- Organiser les étapes d'un projet.
- Prioriser les informations et les actions.
- Faire le lien entre besoin, production et rendu final.

## À relier

Ne pas revendiquer de résultats chiffrés sans preuve documentée.
