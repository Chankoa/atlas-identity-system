# Direction d'image

## Intention

L'image doit soutenir une identité professionnelle sobre, éditoriale et contemporaine.

## Mots-clés

- culture ;
- transmission ;
- clarté ;
- matière éditoriale ;
- lieux ;
- publics ;
- détails ;
- gestes de conception.

## À privilégier

- compositions aérées ;
- lumière naturelle ou douce ;
- textures papier, écrans, carnets, lieux culturels ;
- scènes de travail ou de médiation non artificielles ;
- cadrages précis.

## À éviter

- images trop corporate ;
- portraits fictifs ;
- effets trop spectaculaires ;
- imitation d'une identité visuelle existante ;
- visuels qui remplacent le contenu au lieu de l'accompagner.
