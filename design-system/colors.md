# Couleurs

Palette de base issue de la première direction visuelle du projet.

## Couleurs principales

- Texte principal : `#202020`
- Fond clair : `#FAF8F4`
- Bleu grisé : `#667A8A`
- Vert grisé : `#73806C`

## Usage

- Utiliser `#202020` pour le texte courant.
- Utiliser `#FAF8F4` comme fond sobre et chaleureux.
- Utiliser `#667A8A` pour les éléments secondaires, repères ou liens.
- Utiliser `#73806C` pour les accents discrets.

## Règle

La palette doit rester sobre. Elle accompagne le contenu sans devenir décorative.
