# Principes éditoriaux

## Sobriété

Écrire avec précision. Supprimer les formulations décoratives qui n'ajoutent pas d'information.

## Modularité

Chaque bloc doit pouvoir être réutilisé dans plusieurs formats sans dépendre d'un document final.

## Traçabilité

Chaque argument doit pouvoir être relié à une source dans `content/`.

## Adaptation

Adapter la sélection et l'ordre des informations au destinataire, sans modifier les faits.

## Clarté

Préférer une structure lisible :

- contexte ;
- rôle ;
- action ;
- résultat ou apprentissage ;
- lien avec le poste visé.

## Non-invention

Ne jamais inventer une date, un diplôme, un client, une mission, une métrique ou un résultat.
