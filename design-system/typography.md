# Typographie

Direction initiale :

- Titres : Cormorant Garamond
- Texte courant : Source Sans 3

## Intention

Associer une dimension éditoriale, culturelle et contemporaine à une bonne lisibilité.

## Usage

- Cormorant Garamond pour les titres, citations courtes ou ouvertures.
- Source Sans 3 pour les textes, listes, informations pratiques et contenus longs.

## Règles

- Prioriser la lisibilité.
- Éviter les effets typographiques trop décoratifs.
- Maintenir une hiérarchie claire entre titres, sous-titres et texte courant.
